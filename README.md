# jarvis_ai
A private AI assistant built using Ollama and Streamlit, running completely offline with local LLMs. Features a Jarvis-style conversational interface with memory and customizable personality.
JARVIS - Private AI Assistant (Offline)

A personal AI assistant inspired by Jarvis, built using local LLMs with Ollama and a clean UI using Streamlit.

This assistant runs fully offline, ensuring privacy and control while delivering intelligent conversational responses.

🚀 Features
🤖 Jarvis-style intelligent assistant
🔒 Fully offline (no API required)
🧠 Conversation memory (context-aware chat)
⚡ Fast local inference
🎨 Clean chat UI
🛠️ Customizable personality using system prompts

🏗️ Tech Stack
Python
Ollama (Local LLM)
Streamlit (UI)
